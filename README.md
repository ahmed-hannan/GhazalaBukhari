# Ghazala Bukhari – Fashion Design Website

Official website for **Ghazala Bukhari**, showcasing fashion design work, brand identity, and services.
The site is built with **React** and **Vite**, providing a fast and modern frontend experience.

---

## About the Project

This project serves as the digital presence for the fashion designer **Ghazala Bukhari**.
It highlights the designer’s work, collections, and aesthetic while providing a professional online platform for the brand.

The website focuses on:

* Clean and elegant visual design
* Fast performance
* Responsive layouts for multiple devices
* A modern frontend architecture

---

## Tech Stack

This project is built using:

* **React 18**
* **Vite 5**
* **JavaScript (ES Modules)**
* **HTML5**
* **CSS3**

Vite provides fast development and optimized production builds, while React enables modular and maintainable UI components.

## Installation

Clone the repository and install dependencies.

```bash
git clone https://github.com/ahmed-hannan/GhazalaBukhari.git
cd GhazalaBukhari
npm install
```

---

## Running the Development Server

Start the local development server:

```bash
npm run dev
```

The application will run at:

```
http://localhost:5173
```

Hot reload is enabled, so changes will update automatically during development.

---

## Building for Production

To generate an optimized production build:

```bash
npm run build
```

The compiled static files will be created in the **dist/** folder.

---

## Preview Production Build

To preview the production version locally:

```bash
npm run preview
```

---

## Deployment

This is a static frontend project, which means it can be deployed easily on modern hosting platforms such as:

* Vercel
* Netlify
* GitHub Pages
* Cloudflare Pages

Typical deployment involves:

1. Installing dependencies
2. Running `npm run build`
3. Hosting the generated **dist/** folder
